## Бэкенд приложения блога

Позволяет создавать, редактировать и удалять посты,
а также создавать, редактировать и удалять комментарии к ним.

### Как собирать, тестировать, деплоить и использовать приложение

для сборки необходимо выполнить команду mvn clean package

для запуска тестов mvn test

для деплоя приложение на сервере томкат необходимо

- скачать томкат
- запустить томкат
- из папки target проекта скопировать архив ROOT.war
- поместить архив в папку webapps томката, удалив перед этим
директорию root из webapps

для использования приложения необходимо запустить фронтэнд
приложения
