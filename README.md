## Бэкенд приложения блога

Позволяет создавать, редактировать и удалять посты,
а также создавать, редактировать и удалять комментарии к ним.

### Как собирать, тестировать, деплоить и использовать приложение

для сборки необходимо выполнить команду ./gradlew clean bootJar

для запуска тестов ./gradlew test

для запуска приложения необходимо

- перейти в папку build/libs
- выполнить в консоли команду 
java -jar my-blog-back-app-spring-boot-0.0.1-SNAPSHOT.jar

для использования приложения необходимо запустить фронтэнд
приложения
